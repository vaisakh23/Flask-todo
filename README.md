# FlaskTodo

[![License][license-image]][license-url]

Simple to do web application using Flask and SQLite database.

## Installation

clone repo:
```
$ git clone https://github.com/vaisakh23/Flask-todo.git
```
install dependencies:
```
$ pip install -r requirements.txt
```
run the app:
```
$ python run.py
```
open browser:
```
http://127.0.0.1:5000/
```
